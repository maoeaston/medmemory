# Key Decisions（影响后续实现）

- **Repository 接口设计**（`interfaces.ts`）：每表一个 Repo；跨表搜索独立 `SearchRepository`；JSON 字段（allergies/tags 等）TS 层强类型，实现层负责序列化；查询类返回 null/[]，写入类抛错；MVP 不分页
- **sqlite-wasm 走 `sqlite3Worker1Promiser` v2**（worker 模式 + OPFS VFS）；自 2026-04 deprecated 但官方承诺不移除，PoC 阶段足够；不可用时降级 `:memory:`。`@sqlite.org/sqlite-wasm` 类型导出不全，`connection.ts` 用 `as unknown as` 双重断言 + 最小化 `PromiserFn` interface，无 any
- **Repository 删 metadata 行不删原件 Blob**：原件删除由 Domain 层显式调 `StorageAdapter.deleteFile(storage_key)`（已在 `interfaces.ts` 注释声明）
- **Promiser v2 ESM 调用方式**：npm 包 `sqlite3Worker1Promiser` 导出本身就是 v2 工厂函数（返回 Promise<PromiserFn>），不需要也不应该 `.v2()` 属性访问；`.d.mts` 类型声明标错为 v1 factory + `.v2` 重载，运行时不符
